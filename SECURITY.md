# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting Security Issues

This is a documentation-only repository containing a curated list of AI models.
There is no executable code shipped with this project.

### Link Integrity Issues

If you find broken, redirected, or malicious links in this repository:

1. **Check first**: Verify the link is actually broken by visiting it in a browser
2. **Document the issue**: Take a note of:
   - The broken link URL
   - The page/section where it appears
   - The nature of the problem (404, redirect, malicious, etc.)
3. **Report**: Open an issue on GitHub with the details

### Responsible Disclosure

If you discover security concerns related to external links in this documentation:

- **Do not** attempt to exploit any compromised links
- **Do** report suspected malicious links via GitHub Issues
- **Do** provide enough information for maintainers to verify the issue

We appreciate responsible disclosure and will address issues promptly.

### Contact

For security-related concerns, please use [GitHub Issues](https://github.com/shariqmehmood/AI_Models_Matrix/issues).

---

*Note: This repository contains only Markdown documentation. It does not contain any code that processes user input, stores data, or performs any actions that could introduce security vulnerabilities. The primary security concern is the integrity of external links.*
